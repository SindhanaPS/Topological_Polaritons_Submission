Code accompanying manuscript https://doi.org/10.48550/arXiv.2301.03287.

The code 'cav_mol_k_space.py' with:

1) pset=1 and flag1=2 can generate data for Figure 3a-d of the manuscript (for different fp and fm)
2) pset=1 and flag1=1 can generate data for Figure 3e-h of the manuscript (for different fp and fm)
3) pset=1 and flag1=2 can generate data for Figure 4 of the manuscript (for different fp and fm)
4) pset=2 and flag1=2 can generate data for Figure 5c of the manuscript (with fp=0.4 and fm=0.6)
5) pset=3 and flag1=2 can generate data for Figure 5f of the manuscript (with fp=0.3 and fm=0)

maxk should be set to 100 or larger to obtain the correct Chern numbers

The code 'Cav_ex_plotting.py' with flag1=2 for Figures 3a-d, 4, 5c, and 5f. Additionally, with
flag1=1 for Figure 3e-h
